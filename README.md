## üí° Key Concepts

- **Agents**: Autonomous systems that can reason and take actions
- **Tools**: Functions that agents can call to interact with the world
- **Workflows**: Structured sequences of agent actions
- **State Management**: Tracking context across agent interactions
- **Multi-Agent Systems**: Coordinating multiple specialized agents
